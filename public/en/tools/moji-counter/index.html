Character Counter

X/Twitter Ready

Beyond simple character counts — includes X weighted count, line count, URLs, UTF-8 bytes, and platform limit indicators.

What it does

Real-time character count, no-whitespace count, lines, paragraphs, URLs, UTF-8 bytes, and reading time.

X (Twitter) uses weighted count: URLs count as 23 chars, CJK/emoji as 2 chars each. Reply-mode strips leading mentions.

How to use

  1. Paste your text — character counts and URL stats update in real time.
  2. For X posts, watch the "X weighted count" and "X Standard" remaining.
  3. For replies, enable "Calculate as X reply" to strip leading mentions.
